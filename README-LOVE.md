# Running in LOVE

This folder is already structured as a LOVE game project.

## Requirements

- Install LOVE 11.x: https://love2d.org/

## Run

From this folder, use either:

```powershell
love .
```

or on Windows:

```powershell
.\run-love.bat
```

## Package as a `.love` file

Zip the contents of this folder, not the folder itself, and rename the zip file to:

```text
little-lives.love
```

The archive root should contain `main.lua` directly.

## Current controls

- Left click: interact with buttons and locations
- `Esc`: quit
